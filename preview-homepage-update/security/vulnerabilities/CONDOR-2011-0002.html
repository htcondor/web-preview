CONDOR-2011-0001

Summary:

 

In some circumstances, Condor may allow connections from hosts that are specifically blacklisted in the DENY_{X} authorization list specified in the condor_config file, where {X} can be any of the authorization levels READ, WRITE, ADMINISTRATOR, DAEMON, CONFIG, IMMEDIATE_FAMILY, or NEGOTIATOR.

Impact/Consequences:

low

The exploit allows a blacklisted host some access to condor resources. It does not give them any ability beyond what is granted in the ALLOW_{X} lists.
