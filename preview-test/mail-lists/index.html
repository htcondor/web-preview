HTCondor World

HTCondor-world is a mailing list used to announce new versions of HTCondor and other major HTCondor news. HTCondor-world is a VERY low-volume announcements-only list.

You can subscribe, unsubscribe, and browse the archives at the HTCondor-world mailing list page.

HTCondor Users

The HTCondor-users list is meant to be a forum for HTCondor users to learn from each other and discuss using HTCondor. It is an excellent place to ask questions about using and configuring HTCondor from the user community. Our support pages have more information about additional support channels if this forum does not adequately answer your questions.

Please do not post the same issue to htcondor-users and htcondor-admin at the same time.

You can subscribe, unsubscribe, and browse the archives at the HTCondor-users mailing list page. A page to search the archives is available here.

HTCondor Developers

The htcondor-devel list is meant to be a forum for people developing HTCondor to learn from each other and discuss their experiences. It is not meant to be the official support channel for HTCondor. Our support pages have more information about the offical support channels.

The HTCondor team will attempt to monitor the htcondor-devel list and respond to as much as we can, but htcondor-admin@cs.wisc.edu remains the main support address for HTCondor issues. Please do not post the same issue to htcondor-devel and htcondor-admin at the same time.

You can subscribe, unsubscribe, and browse the archives at the HTCondor-devel mailing list page. Additional ways to search and access the archive is available here.

For comments or questions about this website, please email htcondor-admin@cs.wisc.edu
