Neos Migration

Overview

This wiki provide information about NEOS server migration from ANL to WID/MIR. There are 3 main components that we need to migrate.
  1. NEOS Web Portal and Database
  2. NEOS Solvers
  3. NEOS Wiki

Contact Person

HTCondor

  • Greg Thain
  • Thawan Kooburat

ANL

  • Jason Sarich - jason.sarich at gmail.com [NEOS Developer]
  • Sven Leyffer - leyffer at mcs.anl.gov [NEOS Wiki]

Action Items

  • Obtain license for AMPL, GAMS and Intel Compilers
  • Test running all solvers on our site
  • Download NEOS documentation and change link to local site
  • Setup rsync to transfer job archive
  • Obtain and import NEOS database dump

Completed

  • Migrated NEOS Wiki http://neos2.chtc.wisc.edu/NEOS
  • Setup testing NEOS site http://neos1.chtc.wisc.edu/neos/
    • Running solvers: HelloNEOS, MINLP, filter
  • Installed GAMS and AMPL
  • Use HTCondor to manage NEOS daemons

Migration Plan

NEOS Web Portal Migration

Source

  1. Dump NEOS database
  2. Tar job archive area
  3. Gather all solver XMLs

Destination

  1. Setup the NEOS server
  2. Import NEOS database
  3. ? Start NEOS scheduler
  4. Register all solver from XMLs
  5. Extract job archive area
  6. ? Start NEOS web portal

NEOS Solvers

Source

Destination

  1. Import NEOS wiki database
  2. Restore 'wiki' folder
