    Checkpointing

    Checkpointing is self-contained:
    - Code is linked along with Condor libc
    - Condor signals the process to trigger a checkpoint
    - The process executes its own checkpoint and restart
    In Condor, the checkpoint library is statically linked
    - Always loaded at the same address
    The Hijacker checkpoint library is dynamically linked
