Black Hole Conditions

Execute nodes can sometimes become a "black hole" where jobs go to run but never do. This can be avoided by policy , or by testing nodes for the conditions observed below. This list is not exhaustive.

  • Hung NFS mounts - Linux nodes with hung NFS mounts can end up as black holes if the job requires files from an NFS mount
