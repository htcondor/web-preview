Condor Version 6.1.13 released!

June 1, 2000

Associated Link

Condor version 6.1.13 contains full support for Solaris 2.7. In addition, it fixes a number of bugs in version 6.1.12. There are a few remaining known bugs, but they are all very minor. These known bugs are listed in the version history (see below) and will be fixed in the next release. There are a number of new features in 6.1.13, as well.
