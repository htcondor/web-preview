Description

We have a HTCondor YUM repository for installing HTCondor and HTCondor-CE on Red Hat Enterprise Linux and compatible distributions (such as CentOS or Scientific Linux). Please see instructions below for supported platforms.

Repository Installation Instructions

Note: "root" user must be used to perform installation.

  1. Download and install the YUM repository RPM that matches your operating systems. There is a seperate repository for Current and Stable releases of HTCondor.

Redhat Enterprise Linux 8 and compatible

Current Release
  • https://research.cs.wisc.edu/htcondor/repo/current/htcondor-release-current.el8.noarch.rpm
Stable Release
  • https://research.cs.wisc.edu/htcondor/repo/9.0/htcondor-release-current.el8.noarch.rpm
Supported Platforms
  • x86_64

Redhat Enterprise Linux 7 and compatible

Current Release
  • https://research.cs.wisc.edu/htcondor/repo/current/htcondor-release-current.el7.noarch.rpm
Stable Release
  • https://research.cs.wisc.edu/htcondor/repo/9.0/htcondor-release-current.el7.noarch.rpm
Supported Platforms
  • x86_64

Example

Redhat 7: Install HTCondor's current release 
    yum install -y https://research.cs.wisc.edu/htcondor/repo/current/htcondor-release-current.el7.noarch.rp
