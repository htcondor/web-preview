    Condor

    Preparing a Condor job is convenient
    - No re-programming
    - Just re-link the executable to install the remote
    system call and checkpointing library
    - Re-linking is almost automatic via condor_compile
    However, re-linking is not always possible
    - No access to object (.o) files
    - The program is already running!
